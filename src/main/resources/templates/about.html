Użyte technologie:

  • Spring MVC
  • Spring Data
  • Spring Security
  • Thymeleaf
  • Silnik bazy danych - H2
  • Bootstrap 3

Opis strony:

Niniejsza strona została stworzona w technologii Spring. Celem strony jest zaprezentowanie systemu rejestracji oraz logowania użytkownika w bazie danych, a także mechanizmu uwierzytelniania.

Do zabezpieczenia wykorzystano Spring Security.

Hasła są przechowywane w bazie danych danych oraz szyfrowane algorytmem bCrypt.

Do wygenerowania widoku, wykorzystano silnik Thymeleaf oraz bibliotekę Bootstrap 3.

Strona symuluje działanie gry strategicznej.
